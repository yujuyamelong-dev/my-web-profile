# 나만의 웹 이력서 - 로드맵

## 📋 프로젝트 개요

개인 포트폴리오 및 이력서 웹 페이지를 제작하는 프로젝트입니다.
HTML5, CSS3, JavaScript, TailwindCSS를 활용하여 반응형 웹 디자인의 정적 포트폴리오 페이지를 구현합니다.

### 📌 프로젝트 목표
- 나의 경력, 기술, 프로젝트를 효과적으로 소개하는 웹사이트 구축
- 깔끔하고 전문적인 디자인으로 채용담당자에게 긍정적인 인상 제공
- 모든 기기(데스크톱, 태블릿, 모바일)에서 완벽하게 작동하는 반응형 웹
